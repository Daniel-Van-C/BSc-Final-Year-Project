BSc Final Year Project - Daniel Van Cuylenburg.

YouTube video comments analyser. Enter a link, wait for comments to be downloaded, and analyse them using this tool.


FOLDERS
Corpora: Corpora used to train the machine learning models.

StyleSheets: Stylesheets used for the GUI.

Videos: Examples of pre-loaded analysed videos. All new videos will be saved in this folder.


PYTHON FILES
MainWindow.py: GUI of main window.

Menu.py: GUI of menu window.

contractions.py: Expansions of all possible contracted words.

gui.py: GUI controller.

main.py: Main data processing script.

OTHER FILES
API Key: YouTube API key.
